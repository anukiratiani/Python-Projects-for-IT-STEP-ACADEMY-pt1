## Projects / პროექტები

1. **Guess the Number**  
   - Python console game where the user tries to guess a randomly chosen number;
     Python-ის კონსოლური აპლიკაცია, რომელშიც მომხმარებელი ცდილობს გამოიცნოს შემთხვევით არჩეული რიცხვი.
   - Features: username input, max number selection, limited attempts, feedback for guesses.

2. **Translator**  
   - English ⇄ Italian console translator with interactive language switching; 
     ინგლისურ-იტალიური და იტალიურ-ინგლისური კონსოლ თარჯიმანი, რაც საშუალებას გაძლევთ შეცვალოთ ენა.
   - Features: 300+ words in the database, language selection and switching.

3. **Student Management System (OOP)**  
   - In progress / მიმდინარეობს  
   - Object-oriented Python system for managing students and their information.
