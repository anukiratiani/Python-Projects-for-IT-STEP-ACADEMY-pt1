# Student Management System

სტუდენტების მართვის სისტემა არის Python-ზე დაწერილი აპლიკაცია, რომელიც პორტალზე მუშაობის საშუალებას აძლევს ლექტორებსა და სტუდენტებს.

## მახასიათებლები:

### ბაზები
ინფორმაცია ლექტორებზე: lecturers.txt
ინფორმაცია სტუდენტებზე: students.txt

### ავტორიზაცია
სისტემა იყენებს უნივერსიტეტის ელ ფოსტის სისტემას (@nuga.edu.ge). ყველა მომხმარებლის პაროლი არის "1". შესასვლელად საჭიროა ბაზაში არსებული 
სტუდენტის/ლექტორის სახელის მითითება შესაბამისი დომეინით, შემდეგ კი პაროლის შეყვანა. თუ ერთი სახელით რამდენიმე სტუდენტი იძებნება, 
სისტემა აძლევს არჩევანის საშუალებას.

### სტუდენტის პორტალი
სტუდენტები ხედავენ საკუთარ ინფორმაციას:

სახელი და გვარი
პირადი ID
ფაკულტეტი
მიმდინარე ქულა

### ლექტორის პორტალი
ლექტორებს აქვთ ფართო ფუნქციონალი:
- ახალი სტუდენტის დამატება - შეიყვანეთ სტუდენტის სრული ინფორმაცია. სისტემა ამოწმებს ID-ის უნიკალურობას და ავტომატურად ინახება students.txt ფაილში

- სტუდენტის წაშლა - სტუდენტის ბაზიდან წაშლა ID-ის მიხედვით

- ჩემი კურსის სტუდენტები - ავტომატურად აჩვენებს ლექტორის კურსზე დარეგისტრირებულ სტუდენტებს ცხრილური ფორმატით

- ყველა სტუდენტის სია - ყველა სტუდენტის სია მონაცემებით: ID, სახელი, გვარი, ფაკულტეტი და ქულა

- სტუდენტის ქულის განახლება - მიმდინარე ქულის ჩასწორება ახალით სტუდენტის ID-ს მეშვეობით

### ტექნიკური დეტალები
UTF-8 ენკოდინგი: ქართული ტექსტის სრული მხარდაჭერა;
მონაცემთა ვალიდაცია: ცარიელი ველებისა და დუბლიკატების შემოწმება
